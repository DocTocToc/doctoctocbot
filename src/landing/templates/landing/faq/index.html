Comment être retweeté par le bot?
  1. Être {% retweeted_category_or %}
  2. Suivre {% bot_link_text %}
  3. Inclure le hashtag {% hashtag_lst %}
  4. Poser une question (avec un point d'interrogation)
  5. Ne pas mentionner un compte au tout début du tweet (ceci est considéré par Twitter comme une réponse).
  6. Suivre les recommandations de bonne pratique
Si vous pensez que votre tweet aurait dû être retweeté ou que ces conditions ou les recommandations de bonne pratique doivent évoluer, n'hésitez pas à nous contacter via DM ou via notre formulaire de contact. Nos règles tentent de refléter au mieux les usages de la communauté {% hashtag_lst %} et de défendre ses intérêts. Elles sont décidées collectivement et doivent évoluer selon les souhaits de la communauté.
Comment vous contacter via Twitter?

Le compte du bot est mentionné plus d'une centaine de fois par jour car chaque réponse à une question retweetée par le bot entraîne une mention. Il est impossible de vérifier chaque mention pour savoir si c'est une réponse à une question {% hashtag_lst %} ou bien si c'est une question posée à l'équipe {% bot_link_text %}. Nous vous invitons donc à envoyer un DM à {% bot_link_text %}. Si vous souhaitez discuter avec nous publiquement sur votre TL, il suffit d'envoyer un DM avec le lien vers votre question publique. Nous vous répondrons avec plaisir... si nous savons qu'une question nous a été posée!
