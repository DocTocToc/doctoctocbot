Hashtag

Threads (fils)

Une question peut être contenue dans un seul tweet, ou bien elle peut s’étaler sur plusieurs tweets (un fil ou thread). Si je détecte le hashtag et un point d’interrogation dans 1 fil, je retweete le 1er tweet du fil. Il suffit d’écrire le hashtag {% hashtag_lst %} une seule fois dans le fil. Le hashtag peut être dans le premier tweet, dans le dernier tweet ou dans un tweet intermédiaire.

Publicité

J’essaie dans la mesure du possible de ne pas retweeter la publicité.

Retweets

Je ne retweete pas les retweets.

Citations

Je ne retweete pas les citations, c’est à dire des tweets citant un autre tweet. Un retweet avec commentaire est une citation.

Réponses

Je ne retweete pas les réponses. Mentionner un compte au tout début d’un tweet est considéré par Twitter comme une réponse.

Questions ou affirmations?

Je retweete uniquement les questions. Une question doit comporter un point d’interrogation.
